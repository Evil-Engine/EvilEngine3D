# Evil Engine 3D Ai Policy

## 1. What is AI?

* AI is any LLM (Large Language Model) or Image generation tool.

* What does not count as AI is: Autocomplete (Copilot does not count as Autocomplete), Translators (For translating human language).

## 2. General Guidelines

* Image generation tools are NOT allowed, All art must be made by humans.

* LLM's are assistance not a replacement, You're LLM should not replace you, Its called Copilot, Not Pilot.

* If you use an LLM, You should review all the code it generates.

* If you use an LLM, Please specify how and which LLM you used in your PR (Pull request).
