# How to contribute to Evil Engine 3D

## Issues

* Check to see if anyone else reported it, If not create a GitHub Issue.

## Pull Requests

* You can check [Evil Engine 3D Project](https://github.com/orgs/Evil-Engine/projects/1) for PR ideas.

* Joke Pull Requests will have you banned from making PR's.

* Using AI? Read this first [AI Usage Guide](https://github.com/Evil-Engine/EvilEngine3D/blob/main/AI.md).
